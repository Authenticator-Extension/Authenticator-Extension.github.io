一些二步验证程序支持通用备份文件。换句话说，如果你有一个下列程序的备份文件，你也可以使用下列其它程序来使用这个备份文件。

| 应用 | 版本 | 注释 |
| ----------- | ------- | ----- |
| [Authenticator Extension](https://authenticator.cc) | v5.3.0 | 不支持加密的备份文件 |
{: .table .table-striped}

计划支持：

- [Gnome Authenticator](https://gitlab.gnome.org/World/Authenticator)
